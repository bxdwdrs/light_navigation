# 轻导航浏览器扩展

这是一个用于快速访问常用网站的浏览器扩展工具。用户可以通过该扩展快速访问预设的网站，并支持自定义添加、编辑和删除网站链接。

## 功能特性

- **默认网站**：提供一组默认的常用网站。
- **自定义网站**：用户可以添加、编辑和删除自定义网站。
- **拖拽排序**：支持对自定义网站进行拖拽排序。
- **编辑模式**：进入编辑模式后，可以删除或编辑自定义网站。
